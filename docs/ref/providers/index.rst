appsettings2.providers
======================

The :py:mod:`appsettings2.providers` module contains a set of built-in :py:class:`~appsettings2.providers.ConfigurationProvider` implementations for populating configuration data from Command-Line args, Environment variables, JSON, TOML, and YAML.
